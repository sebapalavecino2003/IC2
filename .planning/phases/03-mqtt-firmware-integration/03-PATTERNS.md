**Analog:** `NodeAlert-Firmware/src/managers/task_manager.h` (full file, 62 lines)

**Rationale:** Same role (FreeRTOS task manager). Task_manager.h provides the exact pattern for declaring a manager class that wraps a FreeRTOS task with init/start lifecycle, forward declarations, static task function, and handle storage.

**Class declaration pattern** (task_manager.h lines 21-61):
```cpp

    void init(SensorManager* sm, StateMachine* sm_state, ErrorHandler* eh);
```

- Replace init() params with references to: `QueueHandle_t` (sensor reading queue + command queue), `ErrorHandler*`, `StateMachine*`

- Add MQTT event handler static function alongside the FreeRTOS task static function
- Task function: `static void mqttTask(void* pvParams)` — follows `monitorTask` pattern
- The MQTT FreeRTOS task runs the MQTT loop — it consumes sensor readings from the existing `sensor_queue` and publishes via ESP-MQTT

**BaseCommand pattern** (Django standard — no existing analog in project):
```python
class Command(BaseCommand):
    help = 'Subscribe to MQTT topics and ingest sensor telemetry into the database'

    def add_arguments(self, parser):
        parser.add_argument('--broker', type=str, default=None,
                          help='MQTT broker hostname')
        parser.add_argument('--port', type=int, default=1883,
                          help='MQTT broker port')

    def handle(self, *args, **options):
        broker = options['broker'] or os.environ.get('MQTT_BROKER_HOST', 'mosquitto')
        port = options['port']
        
        self.stdout.write(f'Connecting to MQTT broker at {broker}:{port}...')
        
        client = mqtt.Client()
        client.username_pw_set(
            os.environ.get('MQTT_SUBSCRIBER_USER', 'mqtt_subscriber'),
            os.environ.get('MQTT_SUBSCRIBER_PASSWORD', '')
        )
        
        client.on_connect = self.on_connect
        client.on_message = self.on_message
        
        client.connect(broker, port, 60)
        client.loop_forever()

    def on_connect(self, client, userdata, flags, rc):
        if rc == 0:
            self.stdout.write(self.style.SUCCESS('Connected to MQTT broker'))
            client.subscribe('nodealert/+/telemetry', qos=0)
            client.subscribe('nodealert/+/events', qos=1)
        else:
            self.stderr.write(f'Connection failed with result code {rc}')

    def on_message(self, client, userdata, msg):
        topic = msg.topic
        try:
            payload = json.loads(msg.payload.decode('utf-8'))
        except (json.JSONDecodeError, UnicodeDecodeError) as e:
            self.stderr.write(f'Invalid JSON on {topic}: {e}')
            return
        
        # Route to handler based on topic pattern
        if topic.endswith('/telemetry'):
            self._handle_telemetry(topic, payload)
        elif topic.endswith('/events'):
            self._handle_event(topic, payload)
```

```cpp
#pragma once

#include <cstdint>
#include <cstring>

/* Buffer size: 20 messages (D-20) */
#define MQTT_BUFFER_CAPACITY    20
#define MQTT_MESSAGE_MAX_LEN    256

struct MqttMessage {
    char    topic[64];
    char    payload[MQTT_MESSAGE_MAX_LEN];
    int     qos;
    uint32_t enqueue_ms;       ///< System uptime when buffered
};

class MessageBuffer {
public:
    MessageBuffer();

    /**
     * @brief Enqueue a message (discards oldest if full — D-20)
     * @return true if enqueued, false if buffer empty but discarding
     */
    bool enqueue(const char* topic, const char* payload, int qos);

    /**
     * @brief Dequeue the oldest buffered message
     * @param[out] msg Populated with the oldest message
     * @return false if buffer is empty
     */
    bool dequeue(MqttMessage& msg);

    /** @brief Number of messages currently buffered */
    int count() const { return m_count; }

    /** @brief Is the buffer empty? */
    bool isEmpty() const { return m_count == 0; }

    /** @brief Clear all buffered messages */
    void clear();

private:
    MqttMessage m_messages[MQTT_BUFFER_CAPACITY];
    int         m_head;     ///< Read index (oldest)
    int         m_tail;     ///< Write index (newest)
    int         m_count;    ///< Current number of buffered messages
};
```

**Enqueue with overflow discard** (D-20 — discard oldest when full):
```cpp
bool MessageBuffer::enqueue(const char* topic, const char* payload, int qos)
{
    if (m_count == MQTT_BUFFER_CAPACITY) {
        // Buffer full — discard oldest (advance head)
        m_head = (m_head + 1) % MQTT_BUFFER_CAPACITY;
        m_count--;
    }

    MqttMessage& msg = m_messages[m_tail];
    
    // Bounds-checked copy (following error_handler.cpp pattern)
    size_t topic_len = strlen(topic);
    if (topic_len >= sizeof(msg.topic)) topic_len = sizeof(msg.topic) - 1;
    memcpy(msg.topic, topic, topic_len);
    msg.topic[topic_len] = '\0';
    
    size_t payload_len = strlen(payload);
    if (payload_len >= sizeof(msg.payload)) payload_len = sizeof(msg.payload) - 1;
    memcpy(msg.payload, payload, payload_len);
    msg.payload[payload_len] = '\0';
    
    msg.qos = qos;
    msg.enqueue_ms = (uint32_t)(esp_timer_get_time() / 1000);
    
    m_tail = (m_tail + 1) % MQTT_BUFFER_CAPACITY;
    m_count++;
    return true;
}
```

**Dequeue pattern:**
```cpp
bool MessageBuffer::dequeue(MqttMessage& msg)
{
    if (m_count == 0) return false;

    msg = m_messages[m_head];
    m_head = (m_head + 1) % MQTT_BUFFER_CAPACITY;
    m_count--;
    return true;
}
```

## Shared Patterns

### FreeRTOS Task Declaration Pattern
**Source:** `NodeAlert-Firmware/src/managers/task_manager.h` (lines 21-61), `sensor_manager.h` (lines 33-116)
**Apply to:** `mqtt_manager.h`

All manager classes follow the same pattern:
1. Forward declare dependencies
2. Constructor + `init()` method wiring references
3. `start*Task()` method calling `xTaskCreatePinnedToCore()` with the task name string, stack size in words, priority as raw integer, `this` as pvParams
4. Private `TaskHandle_t` member
5. Private static task function: `static void taskName(void* pvParams)`

### Error Reporting Pattern
**Source:** `NodeAlert-Firmware/src/core/error_handler.h` (lines 39-49), `error_handler.cpp` (lines 29-64)
**Apply to:** `mqtt_manager.cpp`, `message_buffer.cpp`

All firmware components use `source='MQTT'` for MQTT-related errors:
```cpp
error_handler->reportError("MQTT", "Descriptive error message");
error_handler->clearError("MQTT");  // On recovery
```
The error_handler's exponential backoff (1000ms base, 3 retries, 60s cap) from `sampling_config.h` lines 45-46 is reused for MQTT connection retry decisions.

### Bounds-Checked String Copy Pattern
**Source:** `NodeAlert-Firmware/src/core/error_handler.cpp` (lines 43-63)
**Apply to:** `message_buffer.cpp`

```cpp
size_t len = strlen(source);
```

### Django App Pattern
**Source:** `NodeAlert-Backend/core/models.py`, `views.py`, `serializers.py`
**Apply to:** `mqtt_subscriber.py`

Import modules follow the same style: `from django.utils import timezone`, `from core.models import Device, Reading, Event`. The management command uses `self.stdout.write()` and `self.stderr.write()` for logging (not print()) following Django BaseCommand conventions.
